.swiq 파일 포맷

텍스트 파일 하나로 나만의 퀴즈 세트를 만들 수 있어요. 한 줄이 카드 한 장 — 메모장에서 쓰고, 아이폰에서 열면 끝.

One line per card. Write it in any text editor, open it in Swiq.

한눈에 보기

Grammar at a glance 
# 세트 제목            ← 첫 '#' 줄 = 제목 (선택)
> 설명 줄              ← '>' 줄 = 설명 (선택)
// 주석                ← 무시
문장 | O | 설명(선택)
Q: 질문 | A: 답1;답2 | 설명(선택)
질문 | 1)보기 2)보기 3)보기 4)보기 | 정답번호 | 설명(선택)

세 가지 카드 문법

Three card syntaxes — O/X, short answer, multiple choice

O/X 카드

True/false card 문장 | O | 설명(선택)

정답 자리에는 O X 참 거짓 true false를 쓸 수 있어요.

주관식 카드

Short-answer card Q: 질문 | A: 답1;답2 | 설명(선택)

정답이 여러 개면 ;(세미콜론)으로 구분해요. 어느 것을 입력해도 정답으로 인정됩니다.

4지선다 카드

Multiple-choice card (4 options) 질문 | 1)보기 2)보기 3)보기 4)보기 | 정답번호 | 설명(선택)

보기는 1)~4)로 번호를 붙이고, 정답 자리에는 정답 보기의 번호를 적어요.

파일 규칙

File rules & limits
  • UTF-8 텍스트 파일, 확장자는 .swiq
  • 빈 줄은 무시돼요 — 보기 좋게 자유롭게 띄어 쓰세요
  • 카드가 10~200장 인식되면 가져올 수 있어요
  • 파일 크기는 1MB 이하
  • 문항은 500자, 설명은 2,000자까지
  • 중복 문항은 자동으로 무시돼요

앱에서 여는 법

How to open a .swiq file on iPhone
  1. iPhone에서 .swiq 파일을 다운로드하세요 (Safari·메신저·메일 어디서든)
  2. 파일 앱에서 파일을 열거나, 공유 시트를 띄우세요
  3. Swiq를 선택하면 앱이 파일을 읽어들여요
  4. 미리보기에서 카드를 확인하고 저장하면 끝!
